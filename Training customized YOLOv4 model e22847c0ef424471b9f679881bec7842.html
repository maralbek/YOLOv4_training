Training customized YOLOv4 model

  1. It is required to set up GPU drivers first. Instructions from here can be used for easier and cleaner installation in separate CONDA environment.
    1. Create conda environment 
    conda create -n YOLOv4_conda python=3.7 opencv=3

    b. We install the CUDA-Toolkit and the cuDNN library from the nvidia channel with the following terminal command as explained in the CUDA-Toolkit documentation.

    conda install cuda cudnn -c nvidia -n YOLOv4_conda

    c. Then we activate the created environment

    conda activate YOLOv4_conda
  1. Then we need to clone and configure darknet framework
    1. Clone and switch into the repository in the terminal.
    git clone https://github.com/AlexeyAB/darknet.git
cd darknet

    b. Open and edit Makefile. Set GPU, CUDNN, CUDNN_HALF and OPENCV to 1

    c. Line 27-58: Uncomment the compute capability of your GPU (e.g to train with the RTX 5000 GPU, uncomment the following)

    d. Run the build command.

    sudo make

  1. Prepare Dataset for training
    1. Regarding the Darknet documentation, the Dataset should be created in the/darknet/data folder, but can actually be placed anywhere inside the darknet repository folder. There following files should be placed:

    b. obj.names file contains all class names in the dataset. In our test dataset, we use 9 classes with the following content:

    c. obj.data has this information with the paths to the required files. backup directory is required to store the model

    d. train.txt and test.txt has the list of all images used for training and testing. Images in each set should be different

    e. obj_train_data folder has all images and corresponding annotations in txt. format. The annotation format has the following structure with 1 bounding box per line. (widths and heights are in pixels)

    <object-class> <x_center> <y_center> <width> <height>

f. The image annotations are stored in a file with the same filename as the image filename but with .txt ending. Save the annotations in the same folder the images are stored.

  1. Download pretrained model and configuration file
    1. Download the model
    wget https://github.com/AlexeyAB/darknet/releases/download/darknet_yolo_v3_optimal/yolov4.conv.137

    b. First copy the file yolov4-custom.cfg into the dataset folder with the following command.

    cp cfg/yolov4-custom.cfg data/

    c. Then customize the lines of the copied yolov4-custom.cfg as shown in the training manual. Basically just search with the keyword yolo for the three YOLO-layers in the config file. Then in each of the three [yolo] layers, change classes=80 into classes=9 for our nine classes dataset. Above each [yolo] layer is a [convolutional] layer where we change the filters=255 into filters=42 ((classes + 5) * 3). Finally if you want, you can change the image input width and height in the [net] layer by changing width=608 and height=608 into a size which can be divided by 32 (e.g. width=416 and height=416).

  1. Train the model

a. Use the following command to start training

sudo ./darknet detector train /data/obj.data /data/yolov4-custom.cfg /data/yolov4.conv.137 -map
